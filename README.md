# 8RVL201_Meteo
Projet fil rouge par deux

## Types de déplacement :
Classique, Téléportation (flèche du contrôleur droit)

## Interaction :
Sélection avec le contrôleur gauche (en se rapprochant des villes)

Notre application fonctionne avec un système de villes : lorsque vous interagissez avec l’une d’elles, vous obtenez la météo de la ville.
Il y a 5 types de météo : calme, pluvieux, tempête, neige et brouillard.
Interagissez à nouveau pour revenir au temps de base.
